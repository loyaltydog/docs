## Getting an API key

The LoyaltyDog API uses bearer-token authentication. To request a key for your merchant account, [contact us](https://loyalty.dog/contact-us). Once provisioned, you can manage app keys via [`POST /v2/appkeys`](/api-reference).

<Tip>
  Already have a key? Jump straight to the [Quickstart](/quickstart) or the [Authentication](/authentication) guide.
</Tip>
