title: "Zapier"
description: "No-code automations between LoyaltyDog and thousands of other apps."
---

The LoyaltyDog Zapier app lets you trigger LoyaltyDog actions from any of Zapier's 6,000+ supported apps — and vice versa — without writing code.

## Triggers

| Trigger | Fires when |
| ------- | ---------- |
| New customer | A customer is created in any of your programs |
| Points awarded | A transaction credits points to a customer |
| Gift card redeemed | A LoyaltyDog gift card is redeemed |
| Pass installed | A wallet pass is installed by an end user |

## Actions

| Action | Effect |
| ------ | ------ |
| Create customer | Add a new customer to a program |
| Award points | Credit points to an existing customer |
| Issue gift card | Create a new gift card in a program |
| Send wallet pass | Generate and email a wallet pass to a customer |

## Connect

1. From [zapier.com](https://zapier.com/), search for **LoyaltyDog** in the app directory.
2. Click **Connect**.
3. Paste your LoyaltyDog API token and choose the default program (you can override per-Zap).

<Tip>
  The Zapier app uses the same REST API as everything else here, so anything you can do in a Zap, you can also do directly via the [API reference](/api-reference).
</Tip>
